## CodeAlpha_Portfolio

This is my personal portfolio website built using **HTML**, **CSS**, and optional **JavaScript**. It showcases my skills, projects, resume, and contact information.


## Features

- Clean and responsive design suitable for desktop and mobile.
- Sections include:
  - **About Me**
  - **Skills**
  - **Projects**
  - **Experience**
  - **Education**
  - **Contact**
- Smooth scrolling and subtle animations.
- Hover effects on buttons, links, and project cards.
- Fully self-contained (no external deployment used).
